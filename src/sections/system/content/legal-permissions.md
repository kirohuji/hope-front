# 应用权限说明

本说明详细介绍了应用所需的各种权限及其用途，以帮助您了解我们如何使用这些权限来提供更好的服务。

## 1. 必要权限

应用运行必需的基本权限：
- 网络访问权限
- 存储权限（缓存数据）
- 设备信息（设备标识）
- 通知权限（消息推送）

## 2. 功能相关权限

特定功能所需的权限：
- 相机（扫描二维码）
- 麦克风（语音输入）
- 位置（附近服务）
- 通讯录（邀请好友）

## 3. 权限管理

您可以：
- 随时修改权限设置
- 拒绝非必要权限
- 查看权限使用记录
- 撤销已授权限

## 4. 权限使用说明

我们承诺：
- 仅用于声明用途
- 最小化权限范围
- 保护用户隐私
- 定期权限审查
